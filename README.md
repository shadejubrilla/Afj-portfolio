- **About** — Hero with stats, regulatory coverage card
- **Competencies** — EU AI Act, Python security automation, AI adversarial analysis, risk architecture, network forensics, AML/KYC

## Tech Stack

- Vanilla HTML / CSS / JS — no framework
- Fonts: Inter + Space Grotesk (Google Fonts)
- Icons: Font Awesome 6
- Animations: CSS keyframes + Intersection Observer scroll reveals
- 3D tilt: mouse-driven card rotation (desktop only)
